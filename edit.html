🔑 GitHub 토큰 설정

메뉴를 수정하려면 GitHub Personal Access Token이 필요합니다.
한 번 입력하면 이 기기에 저장됩니다.

토큰 발급 방법
  1. github.com → 프로필 사진 → Settings
  2. 왼쪽 하단 Developer settings 클릭
  3. Personal access tokens → Fine-grained tokens
  4. Generate new token 클릭
  5. Repository access: mega-menu 선택
  6. Permissions → Contents: Read and Write
  7. 생성된 토큰 복사 후 아래에 입력
